Introduction

It is a dynamic and interpreted language comparable to Perl, Ruby, Scheme, or Java. There are no type declaration of variables, parameter, functions or methods in source code. As a result, the code is short and flexible.


Python's Notable feautures:

  • Python is case sensitive, and it does not require semicolon at the end of each statement.
  • Uses elegant syntax, so programs are easier to read.
  • It has a large standard library to support many common programming tasks, including connecting to web servers, searching text with regular expressions, reading and modifying files.
  • Its interactive mode makes it easy to test short snippets of code.
  • It can be embedded into an application to provide a programmable interface.
  • It runs in Mac OS, Windows, Linux and Unix.

    • Python source code It uses the “.py” extension. → “hello.py”

    String

    It has a built-in string class name “str.” They are immutable (they cannot be changed.) String Literals in python are surronded either by single quotations marks, or double quotation marks.

    String methods

    They are similar to a function, but runs on an object

  • s.lower(): return a lowercase string = "THIS SHOULD BE LOWERCASE!" print(string.lower())
  • s.strip(): return a string with whitespace removed
  • s.startswith{‘other’}, endswith(‘other’)
  • s.split(): splits the string into substrings

    • List

    A list is a collection which is ordered and changeable.They work similar to strings, and they are written with square brackets. colors=['red','blue','green']

    Sorting

    We can use a sorted function, which takes a list and returns a new list with those elements in sorted order. The original list is not changed.

    The sorted function can use arguments. For example,

    sorted (list,reverse=True)
